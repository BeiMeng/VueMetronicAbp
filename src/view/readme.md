# vue页面按如下格式新建

```vue
<style lang="scss">

</style>

<style lang="scss" scoped>

</style>

<template>
    <div class="demoManage">
        demoManage
    </div>
</template>

<script>
    export default {
        name: "demoManage"
    }
</script>
```

# 约定规则: vue页面名称，顶级标签class名称  以及 name 均 以首字母小写开头的驼峰命名
